module-7 - Python CGI App

Usage

  1. Install the Apache HTTP server.

    • brew install httpd
    • Alternatively, download and install the XAMPP stack.

  2. Move the files in ./module-7/htdocs into your DocumentRoot.

    • On macOS, I created a symlink:
      rm /usr/local/var/www # homebrew puts httpd docroot here
      ln -s $PWD/module-7/htdocs /usr/local/var/www

  3. Enable the CGI module in httpd.conf:

    • <IfModule !mpm_prefork_module>
        # -- uncomment this line --
        LoadModule cgid_module lib/httpd/modules/mod_cgid.so
      </IfModule>
      <IfModule mpm_prefork_module>
        # -- also uncomment this line --
        LoadModule cgi_module lib/httpd/modules/mod_cgi.so
      </IfModule>

  4. Configure httpd.conf to allow executing CGI files:

    • <Directory "/usr/local/var/www/cgi-bin">
        AllowOverride None
        Options ExecCGI
        Order allow,deny
        Allow from all
      </Directory>

  5. Start the Httpd server, and navigate to the page in your web browser.
